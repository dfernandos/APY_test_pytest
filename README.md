# APY_test_pytest

Testando a api https://reqres.in/

Instalar:
Pycharm - `https://www.jetbrains.com/edu-products/download/other-PCE.html`

> Dependecias:
> - Requests: pip `install requests `
> - Pytest: pip `install -U pytest`
> - JsonPath: `pip install jsonPath `

> Rodar os testes:
 `pytest -v -s  `
